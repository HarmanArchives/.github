## About

I am **Harman Deep Singh**, a Full-Stack Developer focused on modern web applications, real-time systems, dashboards, developer tools, and interactive user experiences.

As my GitHub portfolio grew, I wanted my primary profile to showcase only my most significant and actively maintained projects. Rather than deleting older repositories, I created this organization to preserve projects that played an important role in my development journey.

This archive contains learning projects, experiments, prototypes, UI recreations, coding challenge solutions, temporary development repositories, and legacy applications.
